B)  C++ (Robot) Programmers

1)  Install WindRiver
    DVD from FIRST

2)  Install Subclipse
    Subclipse Download Site:  http://subclipse.tigris.org/servlets/ProjectProcess?pageID=p4wYuA
    Workbench update URL:  http://subclipse.tigris.org/update_1.6.x
    NOTE:  Installation instructions are in the C/C++ Programmer's Guide

    SVN Site (code.google.com):

    # Project members authenticate over HTTPS to allow committing changes.
    svn checkout https://kauaibotsfirst2010.googlecode.com/svn/trunk/ kauaibotsfirst2010 --username scottlibert

3)  Install Doxygen and Eclox
    Doxygen:  
    - Download Site:  http://www.stack.nl/~dimitri/doxygen/download.html#latestsrc
    Eclox download site:  http://www.eclipseplugincentral.com/Web_Links-index-req-viewlink-cid-92.html
    Workbench update URL:  http://download.gna.org/eclox/update/
4)  Install updates to FRC software
    - These currently exist in the Distributables\Latest folder
    - The C_CPP_Programming_Guide.pdf file (in the Documentation\Software) folder 
5)  Install Source Code for Latest WPI Library

6)  Synchronize with Latest Source Code in Repository ***** DO THIS *****

7)  When done, commit your changes to the repository

C)  LabView (Dashboard) Programmers

    - These currently exist in the Distributables\Latest folder
